<h1>Descripción</h1>

<p>
- Este proyecto resulta como trabajo práctico de la materia Programación I, en la Lic. en Producción de Videojuegos y Entretenimiento Digital.
  Se trata de un juego desarrollado con Phaser JS, el cual consta de tres niveles desarrollados con TilesMap, en los cuales hay que recoger estrellas y cuidar
  que nos peguen las bombas. El obejtivo del juego es recolectar el mayor numero de estrellas posibles, pero cuidar de que no se acabe el tiempo.
</p>
